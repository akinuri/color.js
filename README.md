# color.js

A tiny library for color management.

**Example usage**

```javascript
var myColor = color("red");

/*	
	myColor = {
		input	: "red",
		type	: "rgb",
		values	: [255, 0, 0],
	}
*/
```

## Methods

| Method | Description | Usage | Output |
| --- | --- | --- | --- |
| hex() | returns color in hex format | color("red").hex() | "#ff0000" |
